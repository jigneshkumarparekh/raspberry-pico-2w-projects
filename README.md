# Raspberry Pi Pico 2w Projects

This repository contains various projects and examples for the Raspberry Pi Pico 2w microcontroller.

## Projects Included

- **pico2w_HBridge_motor** - H-Bridge motor control projects
- **pico2w_led_test** - LED testing and external LED control
- **pico2w_temp_sensor** - Temperature sensor projects
- **pico2w-motion-sensor** - Motion sensor projects

## Getting Started

Each project folder contains its own implementation files and documentation. Navigate to the specific project folder for more details and wiring diagrams.

## Requirements

- Raspberry Pi Pico 2w
- MicroPython firmware
- Appropriate sensors/components for each project

## Usage

Refer to individual project folders for specific instructions and wiring diagrams.
